Slack Bot python -m connectors.slackbot
1
Startup

Load .env, parse args, build SlackBotConfig. Resolve channel names → IDs. Join channels. Start polling (3s) or Socket Mode.

2
Idle / Listening

Bot polls conversations.history every 3s or receives Socket Mode events. Filters out bot messages, subtypes, already-processed messages.

3
User Message
👤

List all files in /workspace

Bot extracts thread_ts (message timestamp). Strips bot mention + trigger prefix. Appends Slack markdown formatting hint.

Posted immediately in thread. This message gets updated during streaming.

Session saved. Lock released. GCS audit logged. Bot returns to idle polling. Next message in same thread resumes session from disk.

📋 Key Differences at a Glance

Aspect CLI / Console Slack Bot
Entry point python -m agent.agent python -m connectors.slackbot
User input Blocking input() in terminal Slack message (polled or Socket Mode)
Streaming display Real-time token-by-token to stdout with ANSI colors Progress message updated every ~2s with last 500 chars
Thinking blocks 💭 Shown in verbose mode Not shown (stripped from output)
Tool call display 🔧 Plan + ⚙️ Exec with args (200 char cap) ⚙️ Tool names in progress message
Final output ⚛️ [Agent] block with ANSI styling Separate Slack message (Slack markdown)
Usage stats Per-stream + per-turn + session totals in terminal Progress message updated to usage footer
Session key Timestamp or --session file path thread_ts (Slack thread timestamp)
Session location sessions/*.session.json sessions/slack/*.session.json
Concurrency Single user, one prompt at a time Per-thread asyncio.Lock; parallel across threads
Cancellation Ctrl+C → partial history saved + sanitized Timeout (300s default)
REPL loop Yes — continuous prompt loop No — event-driven per message
History sanitization Yes — orphaned tool_results stripped No (lock prevents partial states)

🏗️ Architecture: Call Flow

CLI Mode

__main__ (agent.py)
└─ parse_args()
└─ build_agent() ← shared
└─ asyncio.run(run_repl())
└─ check_mcp_reachable()
└─ load_session() ← shared
└─ LOOP:
└─ input("👤 You: ")
└─ agent.iter(prompt)
│ └─ node.stream() → stdout
└─ _finalize_turn()
└─ save_session() ← shared

Slack Mode

__main__ (connectors/slackbot)
└─ SlackBotConfig
└─ SlackBot(config)
└─ SlackClient (polling / Socket Mode)
└─ AgentInvoker
└─ build_agent() ← shared
└─ POLL/EVENT:
└─ _handle_message()
│ └─ send_message("⏳...")
│ └─ invoker.invoke(on_progress)
│ └─ acquire lock
│ └─ load_session() ← shared
│ └─ agent.iter(prompt)
│ │ └─ node.stream()
│ │ └─ on_progress(last 500)
│ └─ save_session() ← shared
│ └─ release lock
└─ update_message(usage footer)
└─ send_message(response)
